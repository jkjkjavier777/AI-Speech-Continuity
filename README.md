# AI Speech Continuity

A research and writing space for studying continuity, dialogue flow, and transformation in AI-assisted speech.

## What this repo is for
- Tracking source notes and ideas.
- Organizing notebook-style markdown drafts.
- Keeping continuity studies easy to review and revise.

## Files
- `Continuity_Study.md`
- `Transformation_Notes.md`
- `README.md`

## How to use
1. Open the markdown files in GitHub or locally.
2. Read the notes in order.
3. Revise or expand the ideas as the study evolves.

## Status
Active draft repository for ongoing exploration.
